### Projects

**Refractor Radar** — AI Code Analysis System  
Extracts AST structures, embeds functions via Gemini, detects semantic duplication through clustering, and recommends refactoring with diff analysis across large codebases.  
`CrewAI` `ChromaDB` `Gemini` `NetworkX`

**PregPose Pal** — Pregnancy Posture Safety  
Real-time posture classification from phone sensor data. 90% accuracy across 8 posture classes, 30% fewer false positives on high-risk movements.  
`PyTorch` `FastAPI` `OpenCV` `Sensor Data`

**GameAble AI** — Hands-Free Gaming · *1st Place @ 3PercentHacks*  
Desktop app using real-time pose estimation as a game controller. AI agent generates custom pose models per game.  
`Computer Vision` `Pose Estimation` `AI Agents`
